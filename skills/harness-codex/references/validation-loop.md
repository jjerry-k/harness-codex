# Validation Loop

기본 검증 루프는 다음을 포함합니다.

## 1. 산출물 확인
- 요청한 결과가 실제로 만들어졌는가
- 출력 형식이 맞는가
- 범위 밖 변경이 들어갔는가

## 2. 요구사항 대조
- 핵심 요구사항이 빠지지 않았는가
- 숨은 제약을 위반하지 않았는가

## 3. Reviewer 검토
- 논리 오류
- 예외 케이스 누락
- 과도한 복잡성
- 테스트 부족

## 4. 수정 루프
- 피드백 기준 수정
- 수정 후 최소 재검증

## 5. 종료 조건
- 요구사항 충족
- 치명 리스크 없음
- 남은 리스크는 명시됨
